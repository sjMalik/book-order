# book_order

## If you have vue 2 then uninstall it

```
npm uninstall -g vue-cli
```

## Install Vue 3

```
npm i -g @vue/cli
```

## Project setup

```
vue create book_order
npm install
```

### Compiles and hot-reloads for development

```
npm run serve

```
npm run build
